🎵 WebSong – Glassmorphism Music Player

A modern, aesthetic web-based music player built using HTML, CSS, and JavaScript featuring a glassmorphism UI design with background artwork and synced lyrics display.
📌 Project Overview ::-

->>WebSong is a lightweight front-end music player that:

  >Displays animated song lyrics

  >Shows album artwork

  >Includes play/pause & skip controls

  >Displays song duration & progress

  >Uses a beautiful glassmorphism card UI

  >Runs directly in the browser (no backend required)

  🖼️ UI Preview::-

  >Full-screen background artwork

  >Blurred glass-style music player card

  >Center play button with modern controls

  >Lyrics displayed on the left side
  
  >Song progress bar with timer

🚀 Features::-

✅ Play / Pause functionality

✅ Forward & Backward controls

✅ Song duration & live progress tracking

✅ Animated background

✅ Glassmorphism effect

✅ Responsive design

✅ Custom song & image support

🛠️ Tech Stack

  >HTML5

>CSS3 (Glassmorphism + Flexbox)

>JavaScript (Vanilla JS)

>HTML5 <audio> API
